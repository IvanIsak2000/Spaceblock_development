# Spaceblock

Self-hosting social network with clear public API 
(project is currently *in development*)

![home page](docs/landing.png)

## Usage HOWTO


### client.py

```bash
python3 client.py <your user_name> <your user_token>
```

### server.py
```bash
uvicorn server:app --reload
```
